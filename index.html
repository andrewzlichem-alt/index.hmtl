Geometry & Number Theory

Area-2

Ratio of Areas

$$\frac{BP}{CP} = \frac{[\triangle ABP]}{[\triangle ACP]}$$ $$\frac{AP}{PE} = \frac{[\triangle ABC]}{[\triangle BCD]}$$ $$\frac{CP}{PE} = \frac{[\triangle ABC]}{[\triangle DAB]}$$
  • $\triangle ABC$ is equilateral. Point $P$ is somewhere inside the triangle.
  • distance from $P$ to $AB$, $BC$, $CA$ = $\text{length of altitude}$.

FLT and ETT

  • FLT: $a^{p-1} \equiv 1 \pmod p$ ($a$ cannot be divisible by $p$, $p$ is prime).
  • ETT: $a^{\phi(m)} \equiv 1 \pmod m$ where $\lambda(m) \le \phi(m)$. $\lambda(m) = a^k \equiv 1 \pmod m$. If $\gcd(a, m) = 1$, then $a^{\lambda(m)} \equiv 1 \pmod m$.

Trigonometry, CRT, & Graph Theory

Trigonometry

  • Law of Sines: $$\frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C} = 2R \quad (R = \text{circumradius})$$
  • Law of Cosines: $$c^2 = a^2 + b^2 - 2ab \cos C$$

Graph Theory

  • Eulerian Path: Traverses every edge exactly once. Has an Eulerian path iff $0$ or $2$ vertices have odd degrees.
  • Eulerian Cycle: Traverses every edge exactly once. Has an Eulerian cycle if every vertex has an even degree.

CRT (Chinese Remainder Theorem)

  • $m_1, m_2, \dots, m_k$ are pairwise relatively prime.
  • System: $n \equiv a_i \pmod{m_i}$. Has a unique solution $\pmod M$ where $M = m_1 m_2 \dots m_k$.
  • General formula: $$n = \sum_{j=1}^{k} a_j M_j (M_j^{-1} \pmod{m_j}) \pmod M$$

Analytic Geometry, Complex Numbers, & Markov Chains

Analytic Geometry

  • Distance from point $(x_0, y_0)$ to line $ax + by + c = 0$: $$d = \frac{|ax_0 + by_0 + c|}{\sqrt{a^2 + b^2}}$$
  • Shoelace Formula: $$\text{Area} = \frac{1}{2} \left|(x_1y_2 + x_2y_3 + \dots + x_ny_1) - (x_2y_1 + x_3y_2 + \dots + x_1y_n)\right|$$

Multiplicative Order

  • $\text{ord}_m(a)$ is the smallest $k$ such that $a^k \equiv 1 \pmod m$. $\text{ord}_m(a) \mid \phi(m)$.

Complex Numbers

  • Roots of Unity: $z^n = 1$. $z = e^{i\left(\frac{2k\pi}{n}\right)} = \text{cis}\left(\frac{2k\pi}{n}\right)$.
  • Cyclotomic Polynomial: monic polynomial whose roots are primitive $n$-th roots of unity. Degree = $\phi(n)$.

States and Markov Chains

  • State Diagrams: $E_0 \rightarrow E_1 \rightarrow E_2$. Probabilities add up to $1$. Write each state as a product of another state.

Systems of Equations & Inequalities

Linear Systems

  • Unique solution if equations are concurrent, independent, and consistent. Use symmetric properties or Vieta's.

Generating Functions

  • Ordinary: $A(x) = \sum_{n=0}^{\infty} a_n x^n$
  • Convolution: $C_n = \sum_{i=0}^{n} a_i b_{n-i}$
  • Closed forms: $\frac{1}{1-x} = 1 + x + x^2 + \dots$ and $\frac{1}{(1-x)^{k+1}} = \sum_{n=0}^{\infty} \binom{n+k}{k} x^n$

Inequalities

  • Weighted AM-GM: $$\frac{w_1x_1 + \dots + w_nx_n}{w_1 + \dots + w_n} \ge \sqrt[w_1+\dots+w_n]{x_1^{w_1}\dots x_n^{w_n}}$$
  • Cauchy-Schwarz: $$(x_1^2 + \dots + x_n^2)(y_1^2 + \dots + y_n^2) \ge (x_1y_1 + \dots + x_ny_n)^2$$

Advanced Counting & Group Theory

Advanced Counting

  • Roots of Unity Filter: $$\sum_{k=0}^{\infty} a_{kn} x^{kn} = \frac{P(1) + P(\omega) + \dots + P(\omega^{n-1})}{n}$$
  • Burnside's Lemma: $$|X / G| = \frac{1}{|G|} \sum_{g \in G} |X^g|$$

Group Theory

  • Group definition: Closure, Associativity, Identity ($e$), Inverse ($a^{-1}$).

Diophantine Equations

  • Chicken McNugget Theorem: Max non-expressible = $ab - a - b$ for coprime $a, b$.
  • $a^2 + b^2$: No primes congruent to $3 \pmod 4$ raised to an odd power.

Primitive Roots and LTE

  • Primitive root: $\text{ord}_m(g) = \phi(m)$. Exists if $m = 2, 4, p^k, 2p^k$.
  • LTE Lemma: If $p \nmid x - y$, then $v_p(x^n - y^n) = v_p(x - y) + v_p(n)$

3D Geometry

  • Euler's Formula: $F + V = E + 2$
  • Right Cone: $\text{Volume} = \frac{1}{3} \pi r^2 h$, $\text{Surface Area} = \pi r^2 + \pi r \sqrt{r^2 + h^2}$
